Collect and track prayer request effortlessly. 

== Description ==

This plugin was created to collect prayer request for church organizations.

The plugin allows you to place a form on your site to collect prayer request and put them into a custom list area. The area will then allow you to sort and organize Published, Pending, and Archived request.

== Installation ==

This section describes how to install the plugin and get it working.

1. Install the plugin via the Plugin >> Add Plugins on you admin dashboard. 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode [prayer_form] on any page
4. Your ready to go!

== Shortcode ==

In order to display the prayer request form, you can include the following shortcode on any of your pages or posts.

[prayer_form]
